Les technos

Nous aimons utiliser les technos les plus modernes.

Node.Js

JavaScript côté serveur, rapide et efficace.

Angular.JS

Framework créé par Google pour faciliter la création d'applications riches.

Symfony 2

Php est le langage le plus utilisé sur internet. Symfony 2 en est un framework qui permet de réaliser des applications complexes dans un cadre efficace et robuste.

Ansible

Ansible est un outil qui permet d'automatiser la configuration et le déploiement sur des machines distantes.

De l'idée au MVP

Nous vous accompagnons pour construire un MVP à partir de votre idée en vous challengeant sur la valeur ajouté et en vous aidant à prioriser les fonctionnalités à implementer.
